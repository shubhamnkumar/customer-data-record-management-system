# Customer Data & Record Management System

## Project Overview
This project is an Excel-based customer data and record management system designed to manage customer records, service status, priority tracking, and reporting.

## Features
- Customer record management
- Service status tracking
- Priority-based monitoring
- Dashboard reporting
- Charts and analytics
- SLA tracking
- Data validation and conditional formatting

## Tools Used
- MS Excel
- Data Management
- Reporting & Dashboarding
- Basic SQL Concepts

## Purpose
The project was created to improve workflow monitoring, record accuracy, and reporting efficiency using structured data management techniques.
